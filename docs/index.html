Cookify JS v1.0.0

Cookify JS is an super-extreme light-weight javascript Library to manage cookies. With Cookify, You can manage your cookies easily with our simple cdn. As cookify is written in simple Vanilla JavaScript, this Library never slows down your webpage.

Installation

Adding this Library to you project is super-easy. You can either use cdn by putting the below code to your footer or Download the file from the below button and host it yourself

<script src='https://cdn.jsdelivr.net/gh/Harry260/cookify.js/app.js' >>

Usage

With Cookie JS, you can set a cookie with expiry date, delete a cookie, update a cookie, clear all cookie and get all cookie as object or a specified cookie. Where, update and setting can be done in one method. These functions can be executed in one line!

  2. Getting Cookie

Setting cookie

Cookie.set(COOKIE_NAME, COOKIE_VALUE, COOKIE_EXPIRY)


Setting cookie in cookify is easy. Cookie.set() method is used to set cookie. This method has three parameters - name (string), value (string), expiry date (Integer). Here name is the key (identifier) for the cookie and value is the data stored. After setting cookie, it will return the stored value as object.

//Example of seeting a cookie

var name = ''var cookie_name = 'My_Cookie'; //Cookie name

var cookie_value = 'Hello World'; //Cookie value

var cookie_ex = 29; //Cookie expiry date in days

console.log(Cookie.set(cookie_name, cookie_value, cookie_ex)); // Return { name:My_Cookie, value:Hello World }

Note: If the expiry date is null, it will be automatically set to 365 days

Getting cookie

Cookie.get(COOKIE_NAME)


Getting cookie with Cookify only require the cookie name (string) for accesing the value. If the cookie name is wrong, it returns undefined

//Example of geeting a cookie

var cookie_name = 'My_Cookie';

console.log(Cookie.get(cookie_name)); // Return Hello_World

Deleting cookie

Cookie.delete(COOKIE_NAME)


Deleting cookie with Cookify only require the cookie name (string) for deleteing it completely. If the cookie name is wrong, it does nothing.

//Example of Deleting a cookie

var cookie_name = 'My_Cookie';

console.log(Cookie.delete(cookie_name)); // Return 'success'

Clear cookie

Cookie.clear()


With clear cookie.clear() method, all the cookie saved with cookify can be removed with one line of code. It returns success if the operation was successful. This method requires only one parameter which is the name (string) of the cookie.

//Example of clearing cookies

console.log(Cookie.clear()); //No parameters Return

Get all cookies as object

Cookie.obj()


The Cookie.obj() method can be used for getting all the cookies saved as object.This method dont need any parameters

//Example of getting all cookies as object

console.log(Cookie.obj()); //Return (example) { name:'harry', roll-no:10, status:0}
